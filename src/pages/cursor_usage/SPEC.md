# Cursor Usage Dashboard

## 概要
Cursor AIツールの利用状況を追跡するダッシュボードです。コスト、トークン消費量、リクエスト数などのメトリクスを可視化し、AIモデルごとの利用傾向を把握できます。

## データソース
- cursor-usage データセット

## フィルタの使い方

### 日付範囲
利用データの対象期間を指定できます。日付範囲を絞り込むことで、特定期間のコストやトークン消費量を分析できます。

### AIモデル選択
分析対象のAIモデルを選択できます。複数のモデルを比較したり、特定のモデルに絞り込んで詳細を確認できます。

## KPIカード

### Total Cost
選択した期間・モデルでの総コスト（USD）を表示します。

### Total Tokens
選択した期間・モデルでの総トークン消費量を表示します。

### Request Count
選択した期間・モデルでのリクエスト総数を表示します。

## チャート・テーブルの見方

### 日次コスト推移（折れ線グラフ）
日ごとのコスト変化を時系列で表示します。コストの増減傾向や異常値を確認できます。

### モデル別トークン効率（棒グラフ）
各AIモデルのトークン消費量を比較します。どのモデルが多くのトークンを使用しているかを把握できます。

### モデル別コスト割合（円グラフ）
総コストにおける各モデルの割合を表示します。コストの内訳を視覚的に理解できます。

### 詳細データテーブル
日付、モデル、コスト、トークン数などの詳細データを一覧表示します。個別のリクエストレベルでデータを確認できます。
